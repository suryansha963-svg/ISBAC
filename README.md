## ✏️ How to Add a New Video
1. Upload video to YouTube → set to **Public**
2. Copy the video ID from the link (e.g. `youtube.com/watch?v=`**`VIDEOID`**)
3. Open `index.html` in Notepad
4. Find `<!-- VIDEO 1 — Replace YOUTUBE_VIDEO_ID below -->`
5. Remove the `<!--` and `-->` around the video block
6. Replace `YOUTUBE_VIDEO_ID` with your actual video ID
7. Save and re-upload `index.html` to this repository

## ✏️ How to Add a New PDF
1. Upload PDF to Google Drive → Share → **"Anyone with the link"** → Copy link
2. Open `index.html` in Notepad
3. Find `<!-- DOCUMENT 1 — Replace the link below -->`
4. Remove the `<!--` and `-->` around the document block
5. Replace `PASTE_GOOGLE_DRIVE_LINK_HERE` with your Google Drive link
6. Save and re-upload `index.html` to this repository
